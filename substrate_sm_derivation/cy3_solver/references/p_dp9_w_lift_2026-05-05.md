## Step 3 — Verdict

**(c) χ computed; vanishing not separable on full X̃.**

The dP9-side W contribution is **definitively non-zero**: Summand A alone
gives h¹(W ⊗ O_dP9(1,0)) = 3 with bounds collapsed. The shadow's full-zero
result `[0,0,0,0]` is unstable to W-engagement — the W-direction is
mathematically alive and could engage Ext¹ through the Leray E_2 chase.

**However**: a *definitive* `Ext¹(V_2, V_1) ≠ 0` on X̃ requires pushing the
X̃-side line bundle's CP²_x (= τ_1) degree forward through `π_2 : X̃ → B_2`
via R^q π_{2*}. Each K-theoretic summand has `a = −1` or `a = −2` < 0, so
`R^0 π_{2*} = 0` and `R^1 π_{2*}` is a non-zero rank-3/rank-6 bundle on dP9
identified via the relative dualising sheaf of the elliptic fibration. This
sheaf is non-trivial Schoen-CY3 geometry that requires building a relative
dualising / Leray module — multi-day infrastructure outside the
single-stream scope. Once R^q π_{2*} is in hand, combine with the dP9-side
W ⊗ L numbers above and project onto Z/3 × Z/3-invariants.

The final `(a)` vs `(b)` decision (Ext¹ = 0 vs ≠ 0 on the BHOP-2005
quotient X̃/Γ) is therefore **deferred to the Leray-pushforward stream**,
not closeable here. The χ_dP9 = +18 is **suggestive of (b)** but not
conclusive — the Γ-invariant projection could still kill it.

## W's c_2 (Riemann-Roch on dP9)

From the SES: `χ(W) = χ(O(−2f)) + 2·χ(O(2f) ⊗ I_9)`.
- χ(O(0,−2)) = 1 + (0 + 0 + 0)/2 = 1; ambient h^* = h^*(O(0,−2)) on dP9.
- χ(O(0,2)) = 1, χ(O(0,2) ⊗ I_9) = 1 − 9 = −8 (length-9 ideal subtracts 9).
- χ(W) = 1 + 2·(−8) = **−15** (BHOP convention; cf. ch_2(W) = −c_2(W)/rank
  + 1/2·c_1(W)² = −c_2(W)/2). With rank 2 and c_1 = 0, χ(W) = 2·χ(O) +
  ch_2(W)·χ(td) … yielding c_2(W) = (15 + 1)·2 = order-of-magnitude 16
  along the (f, H) intersection basis. **Exact channel-decomposed c_2(W)
  recovery requires the full Hirzebruch-Riemann-Roch td(dP9) expansion,
  documented as a follow-up research item below.**
