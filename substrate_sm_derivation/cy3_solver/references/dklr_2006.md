## σ-functional convention

DKLR define the error functional in **Eq. (27)** as an L¹ deviation of the Monge-Ampère determinant ratio from unity, integrated over X with the CY measure dμ_Ω:

```
σ_k = ∫_X | η_k − 1_X | dμ_Ω

where  η_k = det(ω_k) / (Ω ∧ Ω̄)
```

This is an **L¹ functional**, normalized implicitly by the requirement that ⟨η_k⟩ = 1 after balancing.

The statistical error (Eq. 28) uses an L² variance:

```
δσ_k = (1/√N_p) · [ ∫_X (|η_k − 1| − σ)² dμ_Ω ]^(1/2)
```

So DKLR's σ is L¹ point-wise, with L² statistical error bars from finite sampling.

## Sampling convention

- Points generated via random line intersections with the Fermat quintic (lines drawn in P⁴, intersected with the hypersurface).
- A **mass function** corrects from Fubini-Study measure on P⁴ to the CY measure:

  ```
  m(x) = (Ω ∧ Ω̄) / (i₁* ω_FS)^n
  ```

- N_p **not fixed** in the paper; "one can generate a million points on a quintic in a matter of seconds" — but no specific N_p is quoted with the σ_k results.
- Initialization: balanced metric is sought as a fixed point of the **T-operator iteration**:

  ```
  T(h)_{α β̄} = (N / vol(X)) ∫_X ( s_α s̄_β̄ / ‖s‖²_h ) dμ_Ω
  ```

  Initial h is implicitly the identity (i.e. Fubini-Study Gram).

**Caveat:** The often-quoted "DKLR σ ≈ 0.04 at k=4" is a *figure read*, not a tabulated DKLR value. The actual numerical coefficients (α, β) for the quintic appear later in the literature (see `aklp_2010.md` — Anderson, Braun, Karp, Ovrut 2010 explicitly fit σ_k = 3.51/k² − 5.19/k³ for the Fermat quintic).

## Other quantities

- HYM companion paper (hep-th/0606261, same authors): max[√(1/Vol(Q) ∫_Q (...)² dμ_Ω)] ≈ 0.15 for the Yang-Mills curvature error, and 10 hermitian forms agree to ≈ 0.9% in σ at k=3.
- No Yukawa eigenvalues reported.

## Convention drift to watch out for

- **Some later papers normalize** η_k differently — e.g. dividing by the *integrated* volume rather than the local Calabi-Yau volume form. Always check whether σ is L¹ or L²; DKLR's is L¹.
- The "balanced" metric DKLR converges to is the *Donaldson balanced* metric, not literally the Ricci-flat metric. For finite k there is a residual O(1/k²) gap — that's exactly the σ_k they plot.
